 # Estante Jacomel - Alunos
 
 Interface para que os **alunos** do Colégio Jacomel possam:
 - Pesquisar livros disponíveis na biblioteca
 - Ver status de disponibilidade (disponível / emprestado)
 - Fazer reservas de empréstimo
 
 ## Tecnologias
 - HTML
 - CSS
 - JavaScript puro
 - Hospedagem estática na Vercel
 
 ## Como executar localmente
 1. Clone este repositório:
    ```bash
    git clone https://github.com/hackersolo1/estante-jacomel-real-alunos.git
    ```
 2. Abra o arquivo `index.html` em qualquer navegador moderno.
 
 ## Estrutura básica
 - `index.html` – página principal usada pelos alunos
 - `alunos.css` – estilos da interface dos alunos
 - `alunos.js` – lógica de busca, filtros e interação
 
 ## Deploy
 O deploy deste front-end é feito na **Vercel**, apontando este repositório como origem.
